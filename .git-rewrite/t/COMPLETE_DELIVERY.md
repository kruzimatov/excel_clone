## 🚀 Deployment Instructions

### Prerequisites
- Node.js 16+ installed
- npm or yarn package manager
- Expo CLI installed (`npm install -g expo-cli`)

### Build for iOS
```bash
cd mobile
npm run ios
```

### Build for Android
```bash
cd mobile
npm run android
```

### Web Development
```bash
cd mobile
npm run web
```

### Development Mode
```bash
cd mobile
npm start
```
